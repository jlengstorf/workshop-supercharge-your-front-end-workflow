The lobotomized owl selector makes layout easier. For more info, see the write-up by Heydon Pickering.

*+* margin-top: $vertUnit body margin: 0 padding-top: $vertUnit*2.5

This container wraps all content except the header and footer.

.main-content max-width: $maxWidth width: 100% margin: 0 auto padding-top: $vertUnit*2

.add-columns @include add-columns(3)
